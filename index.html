Which language should you learn?

Python is a strong, highly flexible language. Due to simple syntax, python has a much shorter learning curve than other languages. There are tons of libraries for python, allowing for development and data science tasks with ease. Python is also extensible allowing you to integrate with C/C++ libaries, Java and .NET components. Python also makes Web development a breeze

Ruby is a language designed with the intention of developer happiness and fun. Originating from Japan, Ruby is flexible and easy to get started with. With extensive libraries, you can build a blog engine in 15 minutes! Ruby has a large community and demand for Ruby programmers is increasing - making it a good language to have in your toolkit.

Javascript is a language typically used to make websites responsive. However, Javascript is now able to be used for back end work, such as creating apps and being able to compete with python for doing machine learning! A lot of concepts are transferable to other languages, making it easier to pick up more languages and be more competitive skills.

C# is a productive, high readability language. As a developer a large amount of time is spent reading code, and C# was built with spimplicity and readability in mind. C# is a powerful language and can be used for video games, mobile and desktop applications, and more!
